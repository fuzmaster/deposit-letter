# Marketing Launch Templates

## Template 1 (r/Landlord - Scratching my own itch)

**Title:** I kept messing up the math on security deposit returns, so I built a free local tool to format the letters.

**Body:**
As a landlord, unit turnovers always made me anxious because one sloppy deduction letter can turn into a small claims headache.

I got tired of clunky Word templates, so I coded a simple 1-page web app that handles the math and formats a clean security deposit deduction letter you can print to PDF.

It is 100% free, runs entirely in your browser, and saves zero data.

If anyone wants to try it: https://depositletter.com/

## Template 2 (BiggerPockets Forums - Value-first)

**Title:** Free Tool: A simple Security Deposit Deduction Letter generator.

**Body:**
One of the easiest ways for landlords to lose money is missing the legal return deadline or sending unprofessional itemizations.

I made a lightweight tool that auto-calculates the balance and generates a formal deduction letter you can print as PDF.

It is intentionally simple and browser-based so you can create the document quickly without signing up for anything.

Important reminder: always attach physical receipts/invoices and send via Certified Mail.

Tool link: https://depositletter.com/

## Template 3 (r/SideProject - Builder angle)

**Title:** I built a stateless React app to help landlords avoid small claims court.

**Body:**
I shipped a 1-page Vite + React app for generating security deposit deduction letters.

Main constraint: privacy first. The app is 100% client-side (no backend, no database), so all form data stays local in the browser.

For PDF generation, I skipped heavy libraries and used @media print CSS overrides so the browser handles export natively.

I would love ruthless UX feedback on the form flow before I push this harder.

Project: https://depositletter.com/
